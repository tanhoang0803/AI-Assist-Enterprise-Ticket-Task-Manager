# Roadmap

## Current: Phase 1 — MVP Core

Goal: working full-stack app with auth, basic ticket CRUD, and database.

## Near-term: Phase 2 — Core Product

Goal: daily-usable product with Kanban, notifications, search, and audit trail.

## Mid-term: Phase 3 — AI Layer

Goal: async AI processing pipeline that auto-enriches tickets on creation.

## Later: Phase 4 — Smart Integrations

Goal: Google Calendar sync, file attachments, optional Stripe demo.

## Future Backlog (not planned)

- Real-time collaboration (WebSockets / Socket.io)
- Multi-tenant / workspace model
- Custom workflow automation builder
- Algolia search (replace PG full-text at scale)
- Mobile app (React Native)
- Self-hosted AI model (replace HuggingFace API)
- Microservices migration (if team grows)
- GraphQL API (alternative to REST)
- Advanced analytics dashboard
- SSO (SAML / enterprise login)
